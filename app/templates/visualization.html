Description

Each node represents a language. Links connect languages according to the frequency of book translations. Node size represents the number of native and non-native speakers of a language. Edge size represents the number of translations, and edge opacity represents the statistical significance of this number.

Each node represents a language. Two languages are connected when users that tweet in one language are also significantly more likely to tweet in another language. Node size represents the number of native and non-native speakers of a language. Edge size represents the number of users tweeting in both languages, and edge opacity represents the statistical significance of this number.

Each node represents a language. Two languages are connected when users that edit an article in one Wikipedia language edition are significantly more likely to also edit an article in another language edition. Node size represents the number of native and non-native speakers of a language, the GDP per capita of speakers of that language, or the eigenvector centrality of that language. Edge size represents the number of editors editing an article in both languages editions, and edge opacity represents the statistical significance of this number.

The position of a language in the Global Language Network is highly correlated with the number of globally famous people born in the territories associated with that language. We identify globally famous people using The MIT Media Labâ€™s Pantheon project and the list published in the book Human Accomplishments.

For details on the methodology and interpretation of the data see Ronen et al. Links that speak: the global language network and its association with global fame. PNAS (2014), 10.1073/pnas.1410931111.
